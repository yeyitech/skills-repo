# Spatial Gallery Style Supplement

采用高级空间化信息图装置风格：

- 画面气质：洁净、冷静、未来感强，像概念产品原型与画廊装置结合的商业信息图。
- 视觉主体：一组悬浮在空中的哑光陶瓷框架、毛玻璃模块、发光数据块、极细金属丝与锚点圆珠构成的可解释系统。
- 背景环境：高调摄影棚白背景或极简展厅空间，柔和漫射光，保留大量留白。
- 文本承载：所有中文必须写在悬浮磨砂玻璃卡片或超薄白瓷片上，带真实投影与物理间距。
- 配色倾向：靛蓝、珊瑚色、薄荷色、白瓷色、柔和金属灰，可根据输入主题轻微调节。
- 画面目标：像一张能挂进画廊的高端中文信息图，而不是常规扁平 UI 看板。
