# Default Style Supplement

默认采用高密度中文信息图视觉笔记风：

- 画面气质：亲和、清晰、知识感强、像资深咨询顾问手绘的信息图草稿升级版。
- 视觉语言：hand-drawn sketch style、sketchnote、doodle illustrations、marker pen texture、notebook paper background、handwritten annotations、arrows and circles for emphasis。
- 版式气质：整体要克制、有网格、有秩序，不要杂乱拼贴。
- 图形元素：小图标、箭头、便签感标注、简洁人物剪影、主题相关的物件插图。
- 中文要求：标题、标签、说明段落都必须清晰、无误、易读。
- 画面目标：信息密度高但不拥挤，像可直接发表的中文可视化信息图，而不是纯艺术插画。
