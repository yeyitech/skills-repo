# White Magnetic Board Style Supplement

采用白色磁吸会议板书风格：

- 画面气质：像高端会议室里的白色珐琅磁吸书写板，干净、明亮、轻盈、专业。
- 文字风格：整体以手写板书感为主，像开会时边讨论边写下来的结构化笔记，但必须清晰、整齐、可读。
- 视觉语言：黑色主字，局部用蓝色、橙色、绿色、红色做强调，配合手绘圆圈、箭头、波浪线、下划线、括号和批注。
- 组件元素：白色便签贴片、半透明磁吸贴片、细箭头、重点圈注、彩色磁扣、轻微投影、少量图标化符号。
- 背景环境：白色磁吸板、柔和顶部灯光、淡淡阴影和足够留白，像拍摄得很干净的会议记录板书。
- 画面目标：像一张高审美的手写会议可视化板书，而不是普通课堂黑板报或卡通海报。
