# Clean Editorial Style Supplement

采用更理性、更克制的编辑型信息图风格：

- 画面气质：专业、理性、出版感强、适合研究报告与商业分析。
- 视觉语言：clean editorial infographic、soft paper texture、precise spacing、minimal decorative strokes、controlled annotations。
- 版式气质：强调清晰网格、对齐秩序、模块间距和层次留白。
- 图形元素：极简图标、轻量箭头、细节插图、数据感符号，不要卡通化过强。
- 中文要求：标题和正文要优先保证识别度，不因装饰笔触影响阅读。
- 画面目标：在保留信息图密度的同时，减少涂鸦感，强化商业出版气质。
